About

My name is Benjamin Brody and I am an Information Science major at Northeastern University. I am set to graduate in 2019.I am from Medfield, Massachusetts, and I graduated from Medfield High School in 2014. My interests lie in software development, web design, and database management/creation. I am extremely excited to progress in my skills and look forward to new opportunities and new people.

I have worked in Java, PHP, HTML, and have experience working with a wide variety of people in a number of different workspaces and environments. As far as databases go, I have worked most predominantly in MySQL, but have also worked with MongoDB. My variety in coding languages has offered me a strong foundation with which to learn new languages, and allows me to pick up on certain concepts more quickly. I am eager to learn new skills, and to add to the ones I currently have.

Medfield High School Track Website

Over winter break of 2016-2017, I worked with a friend to develop a track website for our hometown. The wbesite is meant to help coaches, players, and fans view the Medfield track team, and look at their results. The project required a substantial amount of database design when it came to finding ways to represent a track team and its results in MySQL.

The project also called for a certain degree of web design in terms of creating the website and its pages, and representing them in a visually-pleasing and user-friendly manner. As I've never taken any sort of class in web design before, every experience and project I am involved in with the subject is a learning experience.

The source code for this project can be found at my partner's github page here.

  • Date: Winter 2016-2017
  • Subject: Web Development, Database Design

CS3200 Final Project

During the Fall of 2016 a partner and I designed a website for our final project of a database design class. The website is meant to keep track of the office hours for the professors and teaching for a given class.

This project is similar to the track website that I worked on, but was more heavily oriented toward database design. I worked more on the design of the database in this project, and writing procedures and functions to go along with the website to ensure everything worked well with MySQL.

CS3500 Music Editor

For another one of my classes, a partner and I were tasked with creating a music editor in java. The program had to be able to take input in a list of notes, and display the notes in a variety of ways. The program also plays the notes out loud using midi.

This project was quite different from what I had been doing recently. It was much more software oriented and required much more problem solving in terms of data structure and testing. I learned a lot about midi technology and its java integration, as well as jframe, and just how taxing it can be to work with.

The music editor is fully functional as far as the class required it, and it's source code can be found here.
